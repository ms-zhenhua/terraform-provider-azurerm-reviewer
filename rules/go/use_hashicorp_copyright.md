A copyright header is optional. If present, it must be the IBM copyright header.

### good: without copyright header
```go
package example
```

### good: with IBM copyright header 
```go
// Copyright IBM Corp. 2014, 2025
// SPDX-License-Identifier: MPL-2.0

package example
```

### bad:
```go
// Copyright © 2025, Oracle and/or its affiliates. All rights reserved

package example
```
