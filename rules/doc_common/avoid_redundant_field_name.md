Avoid using redundant words in field names.

### good:
```markdown
* `firewall` - (Required) ....
* `subnet` - (Required) ....
* `email` - (Optional) ....
* `endpoint` - (Optional) ....
```

### bad:
```markdown
* `firewall_properties` - (Required) ....
* `subnet_details` - (Required) ....
* `email_address` - (Optional) ....
* `endpoint_configuration` - (Optional) ....
```
